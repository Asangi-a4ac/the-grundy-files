Observations if a=2

For the large category, I followed the convention used in TAKOBEYA, the closed Discord community that is studying this.

Conventions

  • Assume 2 ≤ b ≤ c.
  • Let d=b+2.
  • Let r=c%d.
  • Let q=(c-r)/d. (i.e. Divide c by d, the quotient is q and the remainder is r.)
  • Let // mean the quotient. (Same // as in Python)
  • If multiple sets of conditions are met, the first conditions met should be applied.

Category 1

Conditions

  • c=b+2
  • b%4≠1

Observation

  • Let x_1 = (b-2)//4.
  • Let x_2 = (b-2)%4.
  • The pre-period is nothing.
  • The period length is x_1*8+6+x_2.
  • The period is (0011)^{x_1+1}(2233)^{x_1}(22)(3)^{b-2}.

Category 2

Conditions

  • c=b+2
  • b%4=1

Observation

  • Let x_1 = (b-1)//4.
  • The pre-period is nothing.
  • The period length is x_1*16+6 (=4*b+2).
  • The period is (0011)^{x_1}(0213)(2233)^{x_1-1}(2203)(1001)^{x_1}(1)(2233)^{x_1}(2).

Category 3

Conditions

  • b%4=2
  • c%4=2

Observation

  • The pre-period is nothing.
  • The period length is 4.
  • The period is (0011).

Remarks

The period is equal to gcd(a+b, b+c, c+a).

This might be the easiest one to prove.

Category 4a

Conditions

  • r=2

Observation

  • Let x_1=(b+1)//4.
  • Let x_2=(b+1)%4.
  • The pre-period is nothing.
  • The period length is b+2.
  • Let r=(2) if x_2=1, (22) if x_2=2, and (021) if x_2=3.
  • The period is (0011)^{x_1}r.

Category 4b

Conditions

  • b≠4
  • b%4=0
  • r=d/2+2

Obesrvations

  • Let x_1=b/4.
  • Let y_1=x_1//2.
  • The period length is x_1*2+1.
  • The pre-period length is,
    • If x_1 is even, (8*q+12)*y_1^2 + (2*q+3)*y_1 - 3.
    • If x_1 is odd, (8*q+12)*y_1^2 + (14*q+21)*y_1 + (6*q+7).
  • The period is,
    • If x_1 is even, (12001)(1001)^{y_1-1}.
    • If x_1 is odd, (02)(1001)^{y_1}(1).
  • The pre-period is,
